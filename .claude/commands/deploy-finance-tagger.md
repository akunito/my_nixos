# Deploy Finance Tagger

Rebuild and deploy finance-tagger on VPS from the homelab repo.

**Source of truth**: `~/.homelab/finance-tagger/` on VPS (github.com:akunito/VPS_services.git)
**NOT in dotfiles** — app code lives only in the homelab repo.

## Prerequisites

- VPS is reachable via Tailscale (100.64.0.6, port 56777)
- SSH agent forwarding available (`ssh -A`)

## Steps

### 1. Commit and push in homelab on VPS

If there are uncommitted changes on VPS, commit them first:

```bash
ssh -A -p 56777 akunito@100.64.0.6 '
  cd ~/.homelab &&
  git add finance-tagger/ &&
  git status --short &&
  git diff --cached --stat &&
  git commit -m "feat(finance): <description>

Co-Authored-By: Claude Opus 4.6 <noreply@anthropic.com>" &&
  git push origin main
'
```

### 2. Rebuild container

```bash
ssh -A -p 56777 akunito@100.64.0.6 '
  cd ~/.homelab/finance-tagger &&
  docker compose up -d --build --force-recreate
'
```

### 3. Deploy NixOS config to VPS (only if nix files changed)

Only needed if nix files changed (e.g., homelab-docker.nix, VPS_PROD-config.nix — the `.env` is generated by NixOS activation). Skip if only app code changed.

```bash
ssh -A -p 56777 akunito@100.64.0.6 "cd ~/.dotfiles && git fetch origin && git reset --hard origin/main && ./install.sh ~/.dotfiles VPS_PROD -s -u -d"
```

After NixOS deploy regenerates `.env`, rebuild the container again (step 2).

### 4. Verify

```bash
ssh -A -p 56777 akunito@100.64.0.6 "curl -s http://127.0.0.1:8190/health"
```

Expected: `{"status":"ok"}`

## Development workflow

Edit files directly on VPS via SSH:

```bash
ssh -A -p 56777 akunito@100.64.0.6
cd ~/.homelab/finance-tagger
# edit files...
docker compose up -d --build --force-recreate
```

Or clone the homelab repo locally, push, then pull on VPS.

## Important Notes

- **`.env` is managed by NixOS** activation scripts (`homelab-docker.nix`) — do not edit manually
- **`.env` IS committed** in homelab repo but encrypted via git-crypt — this is safe
- The container healthcheck runs every 30s — give it a moment after rebuild
