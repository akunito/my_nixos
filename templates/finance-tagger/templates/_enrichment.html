How to export from Revolut
  1. Open app.revolut.com and log in
  2. Open DevTools Console (F12 → Console tab)
  3. Copy-paste the export script (revolut-export.js from the repo) and press Enter
  4. Click around in the app (e.g. open transactions or accounts) — this triggers the header capture
  5. Wait for the script to paginate through all pockets (progress shown in console)
  6. A JSON file downloads automatically when done
  7. Upload the downloaded JSON file using the form above

The script intercepts the app’s own API calls to capture auth headers (including the dynamic X-Device-Id), then paginates through all pockets/currencies. Rate-limited to 500ms between requests. Re-uploading is safe — duplicates are skipped.
